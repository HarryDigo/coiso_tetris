Bem vindos ao nosso tutorial!

Durante o período de desenvolvimento do nosso projeto BentoTetris nos deparamos com algumas características essenciais como um estilo para o site (cores), botões estilizados (reação quando o cursor percorre por cima deles) e um outro exemplo são os botões de “início” que estão presentes em todas as páginas do projeto exceto pela inicial, a qual os mesmos retornam. Em outras palavras significa “aperte para voltar ao início”.

Em relação ao jogo:

Um jogo de Tetris possui 7 peças distintas que descem aleatoriamente e infinitamente pela parte superior do tabuleiro quadriculado composto por 20 blocos de altura e 10 de comprimento. As peças são: L; T; I; O; S; Z; J.
O objetivo do jogo é encaixar as peças de forma que se faça um “Tetris!”, para tal, quatro fileiras devem ser completamente empilhadas com um espaço para a peça I eliminá-las simultaneamente. Todas as linhas horizontais que forem completamente preenchidas serão eliminadas (isso é bom).
A fim de que as peças se encaixem, as setas do teclado as movem para a direção que apontam, exceto a para cima: Seta para a esquerda -> move para a esquerda; Seta para a direita -> move para a direita; Seta para baixo -> desce a peça rapidamente; Seta para cima -> gira a peça no sentido horário possibilitando novos formatos de encaixe.

Esse é o básico, divirta-se!
